# Amazon Clone 🛒

A front-end clone of the Amazon homepage built with HTML and CSS.

## 📸 Preview

> A pixel-accurate recreation of the Amazon.com homepage, including the navigation bar, hero section, product category cards, and footer.

## 🚀 Features

- **Responsive Navbar** — Logo, delivery address, search bar, sign-in, orders, and cart
- **Navigation Panel** — "All" menu with Today's Deals, Customer Service, Registry, Gift Cards, and Sell links
- **Hero Section** — Redirect banner (amazon.com → amazon.in)
- **Shop Section** — 8 product category cards including:
  - Eid Special
  - New Year New Supplies
  - Home Essentials
  - Get the Game
  - Electronics
  - Kids Essentials
  - Western Wear
  - Makeup & Skincare
- **Footer** — Multi-column links for Get to Know Us, Make Money with Us, Amazon Payment Products, and Let Us Help You

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure |
| CSS3 | Styling & layout |
| Font Awesome 6.7.2 | Icons (cart, search, location, bars) |

## ⚙️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/amazon-clone.git
   ```

2. **Navigate into the project**
   ```bash
   cd amazon-clone
   ```

3. **Open in browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` to open it in your default browser.

> **Note:** No build tools or dependencies required — this is a pure HTML/CSS project.
